Bem-vindo ao nosso website

Esta é apenas uma versão ALPHA 0.3 do projecto para a disciplina de DAW lecionada pela professora Cristina Vieira. Examplosde possiveis percursos são disponibilizados nesta versão mas apenas por uma questão demonstrativa não sendo ainda funcionais. Este site possivelmente contém bugs e limitações impostas que tornam a utilização do mesmo muito menos intuitiva e interessante do que nós pretendemos com a versão final.

Compatibilidade

O site funciona nos seguintes browsers:

  • Internet Explorer 7+
  • FireFox 10+
  • Google Chrome 17+
